# Statistical Business Analysis Project

## Objective
To analyze sales data using statistical methods and generate business insights.

## Techniques Used
- Descriptive Statistics
- Correlation Analysis
- Hypothesis Testing (3 tests)
- Confidence Intervals
- Linear Regression

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Open notebooks/statistical_analysis.ipynb
3. Run all cells

## Output
- Hypothesis test results saved in reports folder
- Statistical report PDF submitted separately
